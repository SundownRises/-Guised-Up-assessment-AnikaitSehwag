# Domain Glossary — Guised Up

## Core Concepts

### Authenticity Score
A float [0.0, 1.0] computed once at post creation time. Measures how "genuine" a post is based on text heuristics (55%) and image signals (45%). Immutable after creation. Higher = more authentic = ranks better.

### Relationship Depth
A directional, materialized score representing how genuinely user A connects with user B. Stored in the `relationships` table. Incremented by interaction weights (view=1, reaction=2, reply=3). Normalized to [0, 1] using a cap of 100 for ranking purposes. A→B is independent of B→A.

### Semantic Similarity
Cosine similarity between a user's `interest_vector` (384-dim rolling average of interacted post embeddings) and a candidate post's `embedding`. Measures topical alignment. Range [0, 1] in practice.

### Time Decay
Exponential decay function: `1 - e^(-0.02 * age_in_hours)`. Produces a value [0, ~1] that is subtracted from the ranking score. A 7-day-old post is nearly fully decayed. A 1-hour-old post has negligible decay.

### Interest Vector
A 384-dim vector stored on the `users` table. Represents the user's topical interests. Updated on each interaction via exponential moving average: `new = 0.9 * old + 0.1 * post.embedding`. NULL for new users (defaults to 0.5 semantic similarity).

### Candidate Pool
All posts in the system — no pre-filtering by social graph. The ranking algorithm does the filtering, not the query. This avoids the cold-start problem for new users.

### Interaction
A logged event: a user performs an action (view, reply, or reaction) on a post. Stored in `interactions` table. Triggers two side effects: relationship score update and interest vector update.

### Embedding
A 384-dimensional float vector representing the semantic content of text. Generated by `all-MiniLM-L6-v2` via the Python embedding service. Used for search (cosine similarity) and interest modeling.

## Boundaries

### What "Authenticity" Is NOT
- Not engagement (likes/shares/comments are never used for ranking)
- Not popularity (a post with 0 reactions can rank #1 if it's genuine and from a close connection)
- Not manual moderation (it's algorithmically scored, not human-reviewed)

### What "Relationship Depth" Is NOT
- Not a follow/friend graph (you don't "follow" people on Guised Up)
- Not bidirectional (A interacting with B doesn't affect B's depth toward A)
- Not decayed (for MVP — future: daily 0.95 multiplier)
